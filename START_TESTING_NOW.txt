🔧 TROUBLESHOOTING

Problem: Port 5000 already in use
Solution: 
  netstat -ano | findstr :5000
  taskkill /PID <PID> /F

Problem: "No measurements found"
Reason: Normal if Tesseract not installed
Solution: System still works (uses default values)

Problem: Analysis too slow
Reason: First run loads YOLO model (~1.3 seconds)
Note: Subsequent runs will be faster (cached)
